# skills

A collection of agent skills for Claude Code, Gemini CLI, and Codex CLI.

## Skills

### [xr-agent-permissions](./xr-agent-permissions)

Unified permission management for Claude Code, Gemini CLI, and Codex CLI.
Manage allow/deny rules across all three AI coding agents from a single place, with automatic format conversion.

```bash
npx skills add surmrf/skills
```

## Requirements

- Python 3.8+
- One or more of: Claude Code, Gemini CLI, Codex CLI
